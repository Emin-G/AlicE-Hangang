# AlicE - 한강 수온
서울특별시 한강 및 주요지천 수질 측정 자료 API 를 이용하여 한강 수온 정보를 제공 해주는 디스코드 봇

## 필수 모듈
```
"discord.js": "^12.5.3",
"dotenv": "^9.0.2",
"got": "^11.8.2"
```

## 실행 전
.env 파일의
BOT_TOKEN=
SEOUL_API=
를 수정 해 주세요!

## Api 토큰 따는 곳
http://data.seoul.go.kr/dataList/OA-15488/A/1/datasetView.do
